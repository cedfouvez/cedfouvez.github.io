Politique de confidentialité

Dernière mise à jour : 10 février 2026

En résumé : BookLog respecte votre vie privée. Vos données de lecture restent les vôtres. Nous ne les vendons jamais à des tiers.

1. Introduction

CFO Apps ("nous", "notre") développe l'application BookLog. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles lorsque vous utilisez notre application.

2. Données collectées

2.1 Données que vous fournissez

  • Informations de compte : Adresse e-mail et nom d'affichage lors de la création de compte
  • Données de lecture : Livres ajoutés, progression de lecture, notes, avis et statistiques
  • Liste de souhaits : Livres que vous souhaitez lire
  • Données familiales : Si vous utilisez les fonctionnalités de partage familial

2.2 Données collectées automatiquement

  • Données d'utilisation : Interactions avec l'application pour améliorer l'expérience utilisateur
  • Informations techniques : Type d'appareil, version du système d'exploitation, identifiants anonymes

3. Utilisation des données

Nous utilisons vos données pour :

  • Fournir et améliorer les fonctionnalités de l'application
  • Synchroniser vos données entre vos appareils
  • Permettre le partage familial si vous l'activez
  • Vous envoyer des notifications importantes sur le service
  • Analyser l'utilisation de manière agrégée pour améliorer l'application

4. Partage des données

Nous ne vendons jamais vos données personnelles.

Vos données peuvent être partagées uniquement dans les cas suivants :

  • Partage familial : Avec les membres de votre famille que vous avez invités
  • Fournisseurs de services : Firebase (Google) pour l'authentification et le stockage sécurisé des données
  • Obligations légales : Si requis par la loi

5. Stockage et sécurité

Vos données sont stockées de manière sécurisée sur les serveurs Firebase (Google Cloud Platform) avec :

  • Chiffrement en transit (HTTPS/TLS)
  • Chiffrement au repos
  • Authentification sécurisée
  • Règles d'accès strictes

6. Vos droits

Conformément au RGPD et aux lois applicables, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos informations inexactes
  • Supprimer votre compte et toutes vos données
  • Exporter vos données dans un format portable
  • Vous opposer à certains traitements

Pour exercer ces droits, contactez-nous à : cfoappfr@gmail.com

7. Conservation des données

Nous conservons vos données tant que votre compte est actif. Si vous supprimez votre compte, toutes vos données seront effacées dans un délai de 30 jours.

8. Accès à la caméra

L'application utilise la caméra de votre appareil uniquement pour scanner les codes-barres (ISBN) afin d'ajouter facilement des livres à votre bibliothèque.

  • Pas de stockage : Aucune image ou vidéo issue de la caméra n'est enregistrée, stockée ou transmise par l'application.
  • Traitement local : Le scan est effectué localement sur votre appareil pour extraire le numéro ISBN. Aucun flux vidéo n'est envoyé vers nos serveurs ou vers des tiers.

9. Publicités

L'application peut afficher des publicités via Google AdMob. Ces publicités utilisent des identifiants publicitaires anonymes. Vous pouvez désactiver la personnalisation des annonces dans les paramètres de votre appareil.

10. Enfants

BookLog n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans.

11. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications importantes seront notifiées dans l'application. La date de dernière mise à jour est indiquée en haut de cette page.

12. Contact

Pour toute question concernant cette politique de confidentialité ou vos données personnelles :

  • Email : cfoappfr@gmail.com
  • Développeur : CFO Apps
